# Dentelligence — Initial Project Setup

## Create Next.js App

```bash
npx create-next-app@latest ./
```

---

## Initialize shadcn/ui

```bash
npx shadcn@latest init
```

### Recommended Configuration

```text
Component Library: Radix
Preset: Nova (Lucide / Geist)
Style: New York
Base Color: Zinc
CSS Variables: Yes
```

---

## Install Caffeine Theme

```bash
npx shadcn@latest add https://tweakcn.com/r/themes/cmqju1vpt000104kz8qvterg4
```

This updates:

```text
src/app/globals.css
```

and applies the Caffeine color palette.

## Database Setup (Prisma + PostgreSQL)

### Install Prisma

```bash
npm install prisma --save-dev
npm install @prisma/client
```

### Initialize Prisma

```bash
npx prisma init
```

This creates:

```text
prisma/schema.prisma
.env
```

### Configure Database URL

```env
DATABASE_URL="postgresql://username:password@localhost:5432/dentelligence"
```

### Push Schema

```bash
npx prisma db push
```

### Generate Prisma Client

```bash
npx prisma generate
```
